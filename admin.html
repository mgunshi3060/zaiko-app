🔑 GitHub 設定
Personal Access Token (PAT) の取得方法：
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. 「Generate new token (classic)」をクリック
3. repo スコープにチェック → 生成 → コピーしてここに貼り付け
⚠ Tokenはこのブラウザのみに保存されます（GitHubには送信されません）
接続先リポジトリ： mgunshi3060/portal.index (main ブランチ)
保存先： data/docs.json（お知らせ）、data/blogs.json（ブログ）
PDFの保存方法： pdfs/ フォルダに個別保存 → URLをJSONに記録
XSERVERへの移行： JSONファイルをPHPのAPIに置き換えるだけでOK

🌤 ポータル設定
☀️ お天気設定
ホーム画面の日付横に表示されるお天気の取得地域を設定します。
都市名（英語・日本語）または郵便番号で指定できます。
✅ 使用例：Ushiku（牛久市）、Tsukuba（つくば市）、Tokyo、Osaka
※ 日本語でも概ね動作しますが英語のほうが確実です
📅 Googleカレンダー設定
会社共有のGoogleカレンダーを連携すると、一般ページのホームに直近イベントが表示されます。
⚠ カレンダーを 「一般公開」 に設定してから連携してください。
Google カレンダー → 設定 → カレンダーの設定 → アクセス権限 → 「一般公開」にチェック
APIキー取得手順：
1. Google Cloud Console → APIとサービス → 認証情報
2. 「APIキーを作成」→ Google Calendar API を有効化
3. APIキーの制限（推奨）：HTTPリファラー制限でGitHub Pagesのドメインを指定
